Dénombrement : Combinaisons

1) Les combinaisons

Définition : Soient n et k deux entiers naturels tels que 0 ≤ k ≤ n, et E un ensemble fini à n éléments

On appelle combinaison de k éléments parmi n toute partie de E à k éléments
Exemple 1 : On a la possibilité de télécharger 5 films numérotés de 1 à 5.

Le téléchargement des films 2,3 et 5 est une combinaison de 3 éléments parmi 5. C'est le même téléchargement que 2,5,5 ou 5,3,2.
Propriétés : Soient n et k deux entiers tels que 0 ≤ k ≤ n.

Le nombre de combinaisons de k éléments parmi n est égal à : n! / ( n - k )! × k!

Cas particulier : 0!=1 et 0 parmi n = 1 car il n'y a qu'une partie à 0 éléments, c'est l'ensemble vide.
Exemple 2 : Dans un jeu de 32 cartes, une mains est constituée de 5 cartes.

Combien il y a t-il de mains disponibles ? Combien il y a t-il de mains contenant un valet de pique ?

- Dans une main il n'y a pas d'ordre, une main est une combinaison de 5 parmi 32 donc il y a (32×31×30×29×28) / (5×4×3×2×1) = 201'376 mains disponibles.

- On cherche une combinaison de 4 parmi 31 donc (31×30×29×28) / (4×3×2×1) = 31465 mains disponibles.





2) Propriétés des combinaisons

Propriétés : n est un entier naturel.

- 0 parmi n = 1
- 1 parmi n = n
- 2 parmi n = n × ( n - 1 ) / 2
- n parmi n = 1
- Pour tout entier k tel que 0 ≤ k ≤ n, k parmi n = n-k parmi n
Exemple 1 : 7 parmi 10 = 3 parmi 10 car il y a autant de façons de choisir les parties de E à 7 éléments que de partie à 3 éléments.
Relation de Pascal : Pour tout entier n et k vérifiant 1 ≤ k ≤ n-1 avec n ≥ 2, on a : k parmi n = ( k - 1 parmi n - 1 ) + ( k parmi n - 1 )
