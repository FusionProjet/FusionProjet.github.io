Probabilité : Binomial, Bernoulli

1) Épreuve de Bernoulli

Définition : Une épreuve de Bernoulli de paramètre p est une expérience aléatoire à deux issues dont l’une est appelée "succès" et a pour probabilité p, et l’autre appelée "l'échec" de probabilité 1-p

La variable aléatoire qui prend la valeur 1 en cas de succès, et 0 en cas d'échec est appelée variable aléatoire de Bernoulli.

La loi de probabilité de cette variable aléatoire est appelée loi de Bernoulli de paramètre p. Exemple de tableau de Bernoulli

a 0 1
P(X = a) 1 - p p
Exemple 1 : on lance un dé équilibré, le succès est "obtenir un 6".

Il s'agit d'une épreuve de Bernoulli de paramètre p = 1/6



Propriétés : Soit X une variable aléatoire qui suit la loi de Bernoulli de paramètre p. On a :

- L'espérance E(X) = p

- La variance V(X) = p( 1 - p )

- L'écart-type σ(X) = sqrt( p( 1 - p) )
Exemple 2 : Soit X la variable aléatoire qui suit la loi de Bernoulli de paramètre p = 1/6

- E(X) = 1/6

- V(X) = 1/6(1 - 1/6) = 1/6×5/6 = 5/36

σ(X) = sqrt( V(X) ) = sqrt( 5/36 ) = sqrt(5)/6



Définition : Un schéma de Bernoulli de paramètre n et p est une répétition de n épreuves de Bernoulli de paramètre p, identiques et indépendantes.
Exemple 3 : on lance un dé équilibré 2 fois, et on note le nombre de 6 obtenus.

- Il y a répétition d'épreuves de Bernoulli de paramètre p = 1/6 identiques et indépendantes donc on est en présence d'un schéma de Bernoulli de paramètre n = 2 et p = 1/6



Important : phrase à connaître et a récité à chaque fois que l'on croise une répétition d'épreuves de Bernoulli


2) La loi binomial

Définition : On considère un schéma de Bernoulli de paramètre n et p et on note la variable aléatoire égale au nombre de succès au cours de n épreuves

X est donc à valeurs dans { 0 ; 1 ; 2 ; ... ; n }

On dit que la variable aléatoire X suit la loi binomiale de paramètres n et p. On la note B( n ; p )
Propriété : Soit X une variable aléatoire qui suit une loi binomiale de paramètre n et p. Alors la probabilité que X soit égale à k le nombre de succès est : P( X = k ) = k parmis n (nombre de chemin a k succès ) × p k × ( 1 - p )n - k
Exemple 1 : On lance deux fois un dé équilibré. X est la variable aléatoire égale au nombre de 6 obtenus.
- Quelle est la probabilité d'obtenir un 6 exactement ?

Il y a répétition de 2 épreuves de Bernoulli de paramètre p = 1/6 identiques et indépendantes donc on est en présence d'un schéma de Bernoulli de paramètre n = 2 et p = 1/6, X la variable aléatoire égale au nombre nombre de succès suit donc la loi binomiale de paramètre n = 2 et p = 1/6

Donc P(X = 1) = 1 parmis 2 × (1/6)1 × (1 - 1/6 )1
= P( X = 1 ) = 2 × 1/6 × 5/6 = 5/18
