Probabilité : Probabilités indépendantes

1) Les épreuves indépendantes

Définition : Deux expérience sont dites indépendantes si le résultat de l’une n’a aucun impacte sur le résultat de l’autre.
Propriétés : La succession de n épreuves indépendantes d’univers respectifs E1 , E2 , ... , En a pour univers le produit cartésien E1 X E2 X ... X En.

- Les issues sont les n-uplets (x1 ; x2 ; ... ; xn) où x1 appartient à E1, x2 appartient à E2 etc.

- La probabilité d’une issue (x1 ; x2 ; ... ; xn ) est égale au produit des probabilités des issues x1, x2, ... , xn qui la composent.

En autre terme si l'on considère A et B deux événements de deux univers différents p( A∩B ) = p(A) × p(B)
Exemple 1 : On lance un dé non truqué à 6 faces et on note le numéro obtenu, chaque lancer de dés est une épreuve indépendante car la premier lancer n'a aucune conséquence sur le deuxième lancer
Exemple d'exercice : Au bowling, la probabilité que Mike renverse toutes les quilles en un seul lancer de boule est égale à 0,6
Mike effectue 3 lancers successifs, indépendants les uns des autres.

On défini S le succès qui est "Effectue un strike" et T pour "N'effectue pas de strike"

On cherche p(A) = Mike effectue 3 strikes.

donc on cherche p(A) avec A = {SSS}
- p(A) = 0.6×0.6×0.6     car les lancers sont indépendants.
- p(A) = 0.216

La probabilité que Mike effectue 3 strikes en 3 lancers est de 0.216


On cherche p(B) = Mike effectue exactement 2 strikes

Donc on cherche p(B) avec B = {SST ; STS ; TSS}
- p(B) = 0.6×0.6×0.4 + 0.6×0.4×0.6 + 0.4×0.6×0.6
- p(B) = 3×0.62×0.41 = 0.432

La probabilité que Mike effectue exactement 2 strikes en 3 lancers est de 0.432



Astuce : Dans ce genre d'éxercice il est préférable de construire et de s'aider d'un arbre pondéré afin de mieux lire les probabilités ainsi que les issues
